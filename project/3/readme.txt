To get the files in /web/cifar10
    * download the CIFAR-100 python version from https://www.cs.toronto.edu/~kriz/cifar-100-python.tar.gz
    * run convert.py

To get the files in /src/cifar-10-batches-bin
    * download the CIFAR-100 binary version from https://www.cs.toronto.edu/~kriz/cifar-100-binary.tar.gz
